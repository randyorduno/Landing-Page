Oxygen Machine/Oxygen Concentatrator



What does it do?

  • It provides oxygen to patients
  • It maintains patient's oxygen levels to normal
  • It helps them recieve clean oxygen

How to use?

  1. Turn machine on
  2. Wait for yellow light to turn green
  3. Connect nasal cannula into the output area
  4. Place nasal cannula onto patient
  5. Make sure machine is at least 1 ft away from any object

E-Tank/Oxygen Tank



What does it do?

  • It provides oxygen to patients
  • It maintains patient's oxygen levels to normal
  • It helps them recieve clean oxygen
  • It is portable, patients can use it when traveling
  • It can be used when there are blackouts or O2 Concentrator stops working

How to use?

  1. Make sure regulator is placed correctly
  2. Open valve with given key or toggle on E-Tank
  3. Make sure regulatoris set to proper O2 flow.
  4. Place nasal cannula onto patient
  5. Wait until gauge reaches red to switch to another E-Tank

Aspirator/Suction Machine



What does it do?

  • It helps care giver remove any fluids or solids to prevent patient from aspirating
  • It maintains patient's mouth clean
  • It can also be paired with a GOMCO machine

How to use?

  1. Turn machine on
  2. Machine should beging to vibrate
  3. 6ft tubing should be connected to canister
  4. 6ft tubing should also be connected to yankuer on the other end
  5. Use finger to cover the small opening on the yankuer, this is to control when to suction
  6. Use 'U' movements to suction out all fluids/solids
  7. Empty and clean canister after every use

Hand Held Nebulizer/Nebulizer



What does it do?

  • It provides medice that is inhaled
  • It helps the patients whose lungs which are congested for different reason
  • It helps them breathe more confortably

How to use?

  1. Turn air compressor on
  2. Make sure medicine is inside the tubing
  3. Make sure tubing is snugged in all the way
  4. Medicine will begin to nebulize, patient needs to breathe normal
  5. Medicine will take 30 to 45 min to fully nebulize
  6. Once treatment is done rinse tubing with warm water and hang to air dry

Regulator



What does it do?

  • It helps control the amount of oxygen that gets output
  • It lets you measure amount of oxygen left in the tank
  • It helps to make it easier to switch from one tank to the other

How to use?

  1. Make sure regulator is placed correctly onto tank
  2. If gauge reads 2000 psi, it means tank is full. If it reads 1000psi or less, use different tank
  3. Connect nasal cannula into the output area
  4. Place nasal cannula onto patient
  5. Turn knob, it should start at 0 and increase in increments of .5 LPM (Liters per Minute)
  6. Once correct LPM is set, monitor the gauge to see when it reaches 0 psi to switch to a different tank

FEB/Full Electrical Bed



What does it do?

  • It provides a comfort sleep to patients
  • It makes it practical for caregivers to adjust height when needed
  • It allows bed-written patients to sit up when needed, or raise legs when also needed

How to use?

  1. Make sure bed is plugged into socket
  2. Control Box should have a green indicator on to know bed is plugged in
  3. Use controls to adjust bed
  4. Top two buttons adjust the top half to allow the patient to sit or lower to be able to lay down
  5. Middle two buttons allow for entire bed to go up or down
  6. Bottom two buttons alloow bottom half (legs) to be raised or lowered

Half Rails/HSR



What does it do?

  • It prevents patients from falling off the bed

How to use?

  1. Turn knob clockwise to lock onto bed. Counter-clockwise to uninstall
  2. Make sure half rails are installed after the 3rd spring from the head seaction
  3. Use the bar underneath the knob to adjust the height of the rail
  4. There are only 2 positions on the rail
  5. Top position when patient is on the bed. Bottom position when patients needs to get off the bed

Full Ralis/FSR



What does it do?

  • It prevents patients from falling off the bed

How to use?

  1. Side bars need to be installed after the 3rd spring on head section and before 3rd spring on foot section
  2. Full rails will extend to slide in the slots of each sidebar
  3. Once pins are locked on side bar the rails are ready
  4. Top position when patient is on the bed. Bottom position when patients needs to get off the bed
  5. Pull pins on each side to unlock and adjust height

Over Bed Table/OBT



What does it do?

  • It allows bed-written patients to eat comfortably
  • It helps caregivers to place other medical equipment closer to patient

How to use?

  1. Table has two brakes to prevent from moving
  2. Handle underneath table is to adjust height
  3. Clamp on handle to push table into lower height
  4. Lift table to make it taller

E-Cart/ETC



What does it do?

  • It provides a place to hold E-Tank
  • It helps moving tank a lot easier

How to use?

  1. Small knob on the side is to help tighten or loosen the pin that prevents tank from sliding out
  2. Hook is there to hang extra tubing

Nasal Cannula/NC



What does it do?

  • It helps direct oxygen to patients nasal

How to use?

  1. Make sure one end is connected to oxygen output needle
  2. Make sure curved section of the cannula wraps around patients ears
  3. Prongs need to be pointing upward not downward
  4. Cannula is adjustable to have a 'tighter' fit
  5. Prongs can be placed inside a cup of water to test if they are working properly
  6. Bubbles means oxygen is flowing through. No bubbles means it's time to change to a new nasal cannula

Yankuer



What does it do?

  • Paired with the aspirator, helps suction out fluids
  • Small opening helps regulate when to suction
  • When covering the opening, there's suction, otherwise the tip won't suction at all

How to use?

  1. Bottom of the yankuer is connected to 6ft tubing
  2. Tip is used to suction
  3. Vinegar mixed with water can be used to disinfect yankeur
  4. Clean after every use

Nebulizer Mask/ Adult Neb Mask



What does it do?

  • It provides patients the comfort of not having to hold onto tubing during treatment
  • It allows caregivers to give the treatment while patient sleeps
  • It allows patients to inhale treatment through the nose

How to use?

  1. Top half of the nebulizer is connected to the mask
  2. Bottom half is connected to 7 ft tubing
  3. After every treatment, rinse with warm water
  4. IMPORTANT: DO NOT BREAK GREEN PLASTIC THAT IS WHAT NEBULIZES TREATMENT. IF BROKEN OR LOST, TREATMENT WILL NOT BE NEBULIZED
